## Setup backend

1) change this file name .variables.env.tmp to  .variables.env
2) open  .variables.env and paste your MongoDB url here :  DATABASE=your-mongodb-url
3) pnpm install
4) pnpm setup

## Setup frontend
1) cd frontend 
2) pnpm install 

## start server
1) pnpm start

## start react app
1) cd frontend
2) pnpm start

## API Docs 
1) open http://localhost:8888/api-docs in your browser
